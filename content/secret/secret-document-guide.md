# [ 백일몽 주식회사 표준 문서 작성 매뉴얼 ]

본 안내서는 사내망(블로그)에 <strong>새로운 기밀 문서(괴담/탐사 기록)</strong>를 작성할 때 사용하는 전용 서식 및 시스템 제어 기능의 사용법을 정리한 규격서입니다.

---

## 1. 기밀 문서 서식 (`secret-terminal` 숏코드) 사용법

복잡한 HTML이나 자바스크립트 코드를 직접 짤 필요 없이, 본문을 `{ { < secret-terminal > } }` 태그로 감싸기만 하면 **글리치 자물쇠 잠금 화면 ➡️ 해킹 로딩 터미널 ➡️ 순차적 타이핑 및 난독화 효과**가 자동으로 적용됩니다.

### 기본 작성 예시
```markdown
{ { < secret-terminal title="보안관리부<br>어둠 탐사 매뉴얼" > } }
여기에 마크다운으로 문서 본문을 자유롭게 작성합니다.

엔터로 단락을 나누면 각 단락이 시간차를 두고 타닥타닥 타이핑되듯 나타납니다.
* 목록이나 **강조**, [링크] 등 마크다운 문법이 모두 자동 적용됩니다.
{ { < /secret-terminal > } }
```

### 숏코드 옵션 (파라미터)
* `title="..."`: 잠금 화면 및 로딩 터미널에 표시될 문서 제목입니다. (`<br>` 태그로 줄바꿈 가능)
* `obfuscate="true"`: 문서 텍스트 전체에 마인크래프트(`§k`)처럼 글자가 무작위로 계속 변하는 **괴담 난독화 효과**를 부여합니다. (기본값: `false`)
* `hasGame="true"`: 문서를 끝까지 열람한 뒤(약 3.5초 후), 문서가 어둠 속으로 빨려 들어가며 **3D 사무실 탐사 맵**으로 전환되게 합니다. (기본값: `false`)

---

## 2. 어둠 침식 괴담 서식 (`black-shadow` 숏코드) 사용법

평범한 일상 글이나 잡담 게시물처럼 시작하지만, 글을 읽으며 스크롤을 내릴수록 화면 가장자리에 **어둡고 불길한 그늘(그림자)**이 점점 짙어지며 조여오는 침식 연출을 제공합니다.  
복잡한 HTML이나 자바스크립트를 작성할 필요 없이, 본문을 `{ { < black-shadow > } }` 태그로 감싸기만 하면 자동 적용됩니다.

### 주요 기능 및 연출
* **마우스 호버 괴담 텍스트 (`.anomaly-text`)**:  
  본문 중에 `<span class="anomaly-text"><span class="normal">평범한 문장</span><span class="hidden">숨겨진 공포 메시지</span></span>` 서식을 사용하면, 평소에는 '평범한 문장'이 보이다가 마우스를 올리는 순간 붉은색 글리치 효과와 함께 '숨겨진 공포 메시지'(예: "도 망 쳐", "보 지 마")로 변하는 이스터에그를 연출할 수 있습니다.
* **100% 전체화면 3D 어둠 탐사 및 종이 찢기 상호작용**:  
  스크롤이 글의 맨 밑바닥에 도달하면 온 화면이 어둠 속으로 빨려 들어가며 **100% 전체화면 1인칭 3D 사무실 공간**으로 전환됩니다.  
  화면 중앙에 무심하고 건조하게 표시되는 조작법(`[ W, A, S, D ] 이동 | [ 마우스 ] 시선 | [ 클릭 ] 상호작용`)을 참고하여 3D 사무실 안을 탐색하고, **책상 위에 놓인 붉은 조명의 기밀 문서 종이**를 찾아 클릭하세요. 종이를 클릭하면 **사각사각 종이가 찢어지는 효과음과 애니메이션**이 재생되며 어둠 차원이 해제됩니다.
* **평범한 괴담 글로의 정화(복귀) 및 후반부/댓글 해제**:  
  종이를 찢고 원래의 블로그 글로 복귀한 뒤에는 **그늘 침식 오버레이와 바닥의 긴 공백이 영구적으로 해제**되며, 그동안 숨겨져 있던 **글의 후반부 이야기(`<div id="post-after-anomaly">`)와 하단 댓글 창이 부드럽게 나타나** 평화롭고 평범한 일상/괴담 글로 감상할 수 있습니다.

### 기본 작성 예시
```markdown
+++
title = "요즘 방구석에 이상한 그늘이 져"
date = 2026-07-06T20:00:00+09:00
+++

{ { < black-shadow > } }
요즘 들어서 내 방 구석에 <span class="anomaly-text"><span class="normal">자꾸 이상한 그림자가</span><span class="hidden">도 망 쳐</span></span> 지는 것 같아.

밤에 누워있을 때마다 그 <span class="anomaly-text"><span class="normal">그늘이 조금씩</span><span class="hidden">보 지 마</span></span> 내 쪽으로 다가오는 느낌이 들어.

방금도 등 뒤에서 <span class="anomaly-text"><span class="normal">부스럭거리는 소리</span><span class="hidden">뒤 돌아 보 지 마</span></span>가 나서 돌아봤는데 아무것도 없었어.

그런데 화면을 보고 있는 지금, 
모니터에 비친 내 뒤에... 
{ { < /black-shadow > } }
```

## 3. 글마다 선택적으로 켜는 [작가 코멘트] & [댓글] 기능

이제 모든 글에 코멘트나 댓글 창이 강제로 뜨지 않습니다. **원하는 글에서만** 포스트 상단 Frontmatter(메타데이터)에 설정을 추가하여 켤 수 있습니다.

### ① 작가 코멘트 추가 및 제목 커스텀하기
글 상단 Frontmatter에 `authorComment` 또는 `author_comment` 항목을 추가하면 글 하단에 노란색 테두리의 작가 코멘트 영역이 나타납니다.  
기본적으로 **`📝 작가 코멘트`**라는 제목이 표시되지만, 글의 분위기나 콘셉트에 맞게 `authorCommentTitle` 항목을 추가하여 제목 텍스트도 자유롭게 수정할 수 있습니다:
```toml
+++
title = "문서 제목"
date = 2026-07-07T12:00:00+09:00

# 작가 코멘트 제목 커스텀 (옵션, 미입력 시 '📝 작가 코멘트')
authorCommentTitle = "⚠️ 기록관 첨언"

# 작가 코멘트 내용 입력 (마크다운 사용 가능)
authorComment = "여기에 작가 코멘트나 후기를 작성하세요. **굵은 글씨**나 [링크]도 편하게 지원됩니다!"
+++
```

### ② 댓글 창 활성화 및 제목 커스텀하기
Frontmatter에 `enableComments = true` (혹은 `comments = true`)를 추가하면 하단에 댓글 영역이 활성화됩니다.  
기본 제목인 **`💬 댓글`** 또한 `commentTitle` 항목을 추가하여 글의 콘셉트에 맞게(예: "📬 독자 피드백", "💭 방명록") 자유롭게 변경할 수 있으며, 댓글 입력창 내부의 기본 문구는 `commentPlaceholder` 항목으로 원하는 텍스트를 지정할 수 있습니다.

**임의의 가상 기본 댓글은 전혀 노출되지 않으며**, 작성자 본인이나 다른 사람이 작성한 댓글은 모두 **해당 글의 마크다운(`.md`) 파일 Frontmatter(`[[commentList]]`)에 직접 저장되고 렌더링**됩니다. 따라서 댓글을 연출하거나 첨부하고 싶다면 아래와 같이 `[[commentList]]` 구문을 추가하여 관리할 수 있습니다:
```toml
+++
title = "문서 제목"
enableComments = true

# 댓글 창 제목 커스텀 (옵션, 미입력 시 '💬 댓글')
commentTitle = "📬 독자 피드백"

# 댓글 입력창 placeholder 커스텀 (옵션)
commentPlaceholder = "여기에 의견을 기입하십시오..."

# 첫 번째 가상 댓글 설정
[[commentList]]
author = "익명의 탐사대원 #402"
date = "2026-07-07 11:24"
text = "이 매뉴얼 덕분에 어제 F등급 어둠에서 무사히 빠져나왔습니다. 3번 항목 꼭 숙지하세요..."
color = "#666"        # 댓글 왼쪽 세로 포인트 테두리 색상 (기본 회색)

# 두 번째 가상 댓글 설정 (경고/검열 콘셉트)
[[commentList]]
author = "[보안관리부 검열됨]"
date = "2026-07-07 12:01"
text = "⚠️ 보안 프로토콜 위반으로 블라인드 처리된 피드백입니다."
color = "#ff3333"     # 댓글 왼쪽 세로 테두리 색상 (빨간색 경고선)
+++
```

## 4. 🫧 전역 클리커 숨기기 (플로팅 버튼)

화면 우측 하단에 위치한 **`[ 소리: OFF/ON ]`** 버튼 바로 위에 **`[ 클리커: ON/OFF ]`** 버튼이 항상 노출되어 있습니다. 
* 언제든 이 버튼을 클릭하여 화면 상단의 기포(Bubble)들과 좌측의 업그레이드 리모콘 HUD를 즉시 숨길 수 있습니다.
* 숨긴 상태는 브라우저를 닫거나 다른 글을 읽어도 계속 유지되므로, 텍스트 몰입이 필요할 때 유용하게 활용하세요.
